Windows


     
           O windows é um sistema operacional de multitarefas para computadores e dispositivos móveis desenvolvido pela Microsoft. Esse sistema começou a ser projetado em 1981 mas 
          somente em 1993 sua primeira versão foi oficialmente lançada e atualmente é o  sistema operacional mais utilizado. A função do windows é facilitar o acesso do usuário 
          ao computador através de uma interface eficiente, atrativa e fácil de utilizar, além de utilizar softwares mais rápidos e eficazes.

          Atualmente o windows conta com diversas atualizações feitas ao longo do tempo, conforme melhorias na funcionalidade, 
          desing e eficiência foram sendo implementadas no sistema.
          
          Suas seguintes versões são:
          
          
          Windows 1(1982- 1985)
          
          Possuía uma interface gráfica e mouse como grande inovação;
          Possuía elementos gráficos como barra de menu, caixa de mensagem e barra de rolagem;
          Trouxe versões primitivas do paint, bloco de notas e calculadora.
          
          
          Windows 2.0 e 2.11 (1987- 1990)
          
          Capacidade de memória expandida e ícones na área de trabalho;
          Melhora no suporte de memória gráfica possibilitando sobreposição de janelas, controle de dimensões e atalhos de teclado;
          Painel de controle adicionado para uma melhoria no gerenciamento do sistema;
          Versão disponibilizada para  outras empresas conseguirem desenvolver programas nele.
          Windows 3.0 e Windows NT (1990-1994)
          
          Melhoria no desempenho;
          Introdução do gestor de programas, gestor de ficheiros e de impressão;
          Windows NT representa um grande marco para a microsoft ja que foi a conclusão do projeto iniciado nos anos 80 visando a criação de um novo e avançado sistema operativo;
          Windows 95 (1995-1997)
          
          Seu lançamento contou com uma melhora significativa no desing e na usabilidade do software;
          Estreou o menu iniciar, sendo um painel com acesso simplificado às principais funções e aos programas instalados no computador. Posteriormente 
          esse menu foi removido com o lançamento do windows 8, porém a empresa voltou com o recurso com o lançamento do windows 10;
           
          Windows 98, Windows Me e Windows 2000 (1998-2000)
          
          O Windows 98 foi a primeira versão do Windows lançada especificamente para os consumidores finais
          Contou com a abertura e fechamento de programas rapidamente, suporte para leitura de DVDs e dispositivos USB.
          Windows Me, lançado em 14 de setembro de 2000, foi projetado para o uso residencial dos computadores e oferecia diversas melhorias para redes domésticas, músicas e vídeos.
          
          
          
          
          
          
          Entre outras melhorias, o Windows 2000 Professional simplificou a instalação de dispositivos de hardware, ao aumentar a compatibilidade deles com a tecnologia Plug and Play.
          O Windows 2000 Professional, lançado em 17 de fevereiro de 2000, foi projetado para substituir o Windows 95, o 98 e o NT em todos os computadores e laptops de uso profissional
          
          Windows XP (2001-2005)
          
          Possuia uma interface totalmente reprojetada e centrada em um centro de serviços de ajuda e suporte unificados;
          Contou com atualizações frequentes de segurança carregadas da internet.
          
          
          Windows Vista (2006-2008)
          
          No momento, foi reconhecido como o sistema mais seguro disponibilizado pela microsoft por oferecer o recurso de proteção User Account Control que ajudava 
          a prevenir que os softwares possivelmente maliciosos modificassem os computadores;
          O windows vista trouxe melhorias no Windows Media Player devido ao aumento no numero de usuários  que usavam seus computadores como centrai de mídias digitais . 
          Nele era possível assistir televisão, enviar fotografias e editar vídeos.
          Disponibilizou uma nova interface gráfica, chamada de Aero, promovendo que a barra de tarefas e as bordas das janelas recebessem um desing novo .
           
          
          Windows 7 (2009-2012)
          
          Trouxe novas funcionalidades, como modos de trabalhar com janelas, telas sensíveis ao toque e melhoramento na barra de tarefas que incluiam miniaturas das janelas ao vivo.
          
          
           
          
          Windows 8 e 8.1 (2012-2014)
          
          Estreia de uma versão mais funcional e prática que o windows 7;
          Adicionou uma série de gestos baseados no toque ;
          O Windows 8.1 adicionou melhorias nas características básicas, como a busca e a opção de personalização 
          e retornou com o botão iniciar ao canto inferior esquerdo da área de trabalho.
           
          
          Windows 10 (2015)
          
          Ao contrário de edições anteriores, o Windows 10 foi vendido como um serviço;
          Os usuários receberam grandes atualizações que promoveram a adição de novos recursos e tecnologias de forma gratuita e horizontal;
          Apresenta  uma interface desenvolvida para telas mais sensíveis ao toque, com uma abordagem mais convencional o Windows 10 funciona bem em telas touch e desktops.
          
          
          
          Windows 11 ( 2021)
          
          Versão mais atual do Windows, apresenta um desing mais limpo e minimalista ;
          Traz avanços de produtividade, com um modelo especial para gerenciar janelas;
          Apresenta o Auto HDR, que aplica o recurso HDR automaticamente em conteúdo que originalmente não foi desenvolvido com a tecnologia
          Possui novidades como a disponibilização do bloco de notas em modo escuro, microsoft teams integrado, suporte a aplicativos android, recurso de widgets 
          e assistente virtual escondida.
